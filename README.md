# Binance Impulse Alerts

Локальный скринер для Binance USDT Futures.

## Что делает

- Берёт фьючерсные пары Binance с объёмом за 24 часа от `1,000,000 USDT`.
- Следит только за лонговым движением текущей `1m` свечи.
- Если монета проходит импульс от `1.5%`, она попадает в короткое наблюдение.
- Во время наблюдения скринер смотрит стакан и сделки.
- Если под спредом появляется bid-сайз от `20,000 USDT` и его быстро проедают, появляется алерт `INIT LONG`.
- Клик по карточке копирует тикер.
- Кнопка `↗` на карточке открывает мини-график.

## Важные ограничения

Это не торговый бот и не финансовая рекомендация. Данные идут из публичных API/WebSocket Binance, поэтому очень быстрые события могут отличаться по времени от профессиональных терминалов.
